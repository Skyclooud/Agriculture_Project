# 🌾 AgriVision Crop Prediction Dashboard

AgriVision is an AI-powered crop recommendation dashboard built using Python, Machine Learning, and Streamlit. The application predicts suitable crops based on agricultural parameters such as soil nutrients, temperature, humidity, pH level, and rainfall.

The goal of this project is to help farmers, students, and researchers make smarter agricultural decisions using data-driven insights.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Skyclooud/agrivision-crop-prediction-dashboard.git
```

Move into the project folder:

```bash
cd agrivision-crop-prediction-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
